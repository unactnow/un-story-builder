# UN Story Builder

Internal CMS for authoring **feature stories** (HTML export) and **timelines** (standalone timeline HTML export). Content is edited in the admin UI; export produces self-contained HTML using the UN base stylesheet plus merged feature-story and timeline static assets.

## Features

- Block-based **feature story** editor (14 block types) with drag-and-drop reorder, move up/down, revisions, and HTML export
- **Timeline** editor with events, reordering, revisions, and HTML export
- Session-based auth, CSRF protection, admin dashboard with counts and recent items
